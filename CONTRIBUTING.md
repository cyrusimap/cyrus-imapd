# Contributing to Cyrus

So, you want to contribute to Cyrus?  Great!

You should read [our development
process](https://www.cyrusimap.org/dev/developer/process.html) docs.  They're
on the web, but they're also in the repository as
`docsrc/developer/process.rst`.

Go read those!  Duplicating them into this document would just make one more
place we could forget to update.
